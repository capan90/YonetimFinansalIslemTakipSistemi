# Güncelleme Akışı

## Amaç

Uygulamanın yeni sürümlerini kullanıcıya kontrollü ve kolay şekilde ulaştırmak.

## V1 Kararı

- ClickOnce kullanılacak.

## Beklenen Davranış

- Program açıldığında güncelleme kontrolü yapılacak.
- Yeni sürüm varsa kullanıcıya bilgi verilecek.
- Kullanıcı onaylarsa güncelleme başlayacak.
- Uygulama içinde ayrıca “Güncellemeleri Denetle” butonu olacak.

İlerleyen aşamalarda GitHub release süreci ve yayın adımları ayrıca netleştirilecektir.
