# Analiz

## Proje Özeti

Bu proje, işletme içinde kullanılan özel finansal işlemleri kayıt altına almak ve takip etmek için geliştirilecek Windows masaüstü uygulamasıdır.

## Temel İhtiyaç

Sistem; işlem girişleri, kullanıcı hareketleri, kasa bakiyeleri, yetkilendirme, audit log, raporlama, döviz ekranı ve güncelleme sürecini yönetebilmelidir.

## Kullanım Senaryosu

- Uygulama çalışan bilgisayarlarına kurulacaktır.
- Veritabanı merkezi olarak tek bilgisayarda çalışacaktır.
- Tüm kullanıcılar aynı veriyi görecektir.
- Kullanıcılar yönetici tarafından manuel oluşturulacaktır.

## Temel Modüller

- Login
- Kullanıcı yönetimi
- Yetki yönetimi
- İşlem yönetimi
- Raporlama
- Audit log
- Döviz ekranı
- Güncelleme sistemi
- Dialog sistemi
