# Dialog Sistemi

## Amaç

Uygulama içinde standart MessageBox yerine daha kontrollü ve kurumsal görünümlü özel dialog yapısı kullanmak.

## Dialog Tipleri

- Bilgi
- Başarılı
- Uyarı
- Hata
- Soru

## Kurallar

Her dialog aşağıdaki bilgileri içermelidir:
- başlık
- mesaj
- ikon
- renk
- uygun butonlar

## Örnek Kullanımlar

- Kayıt silme onayı
- Kaydedilmemiş değişiklik uyarısı
- Güncelleme bulundu bildirimi
- Başarı ve hata mesajları
