# Yönetim Finansal İşlem Takip Sistemi

Yönetim Finansal İşlem Takip Sistemi, şirket içi kullanım için geliştirilen Windows masaüstü uygulamasıdır.

Bu sistem; finansal işlem girişlerini, kullanıcı yetkilerini, kasa bakiyelerini, raporlamayı, audit log kayıtlarını, döviz bilgilerini ve uygulama güncelleme sürecini merkezi veritabanı üzerinden yönetmek amacıyla tasarlanmaktadır.

## Proje Hedefi

Bu projenin amacı, işletme içindeki özel finansal işlemlerin düzenli, izlenebilir ve denetlenebilir şekilde kayıt altına alınmasını sağlamaktır.

## Temel Özellikler

- Kullanıcı girişi
- Kullanıcı ve yetki yönetimi
- Finansal işlem giriş ekranı
- İşlem listeleme ve filtreleme
- Rapor ekranı
- Audit log sistemi
- Özel dialog sistemi
- Güncelleme kontrol sistemi
- Döviz ekranı
